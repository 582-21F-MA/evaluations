# Examen 2

> Remise : 7 mai\
> Pondération : 10 %

Votre dépôt Classroom contient un programme qui permet aux
utilisateur·rices d'ajouter et de supprimer des items à leur panier.
Votre tâche pour cet examen est double :

1. Réusinez le code de sorte à respecter les [critères d'évaluation].
2. Réglez le bogue lorsqu'on supprime un item inexistant.

[critères d'évaluation]: https://github.com/582-21F-MA/evaluations#crit%C3%A8res-d%C3%A9valuation

## Instructions

1. Commencez par [créer un dépôt Classroom][Classroom] pour l'examen.
2. Ensuite, téléchargez le dépôt sur votre ordinateur (bouton _Code_
   vert → _Download ZIP_).
3. Pour soumettre votre travail, téléversez le fichier « index.js » (et
   seulement ce fichier) dans votre dépôt Classroom. Ne renommez pas le
   fichier, ne le placez pas dans un dossier, et ne le compressez pas.

[Classroom]: https://classroom.github.com/a/4fJdaTsW
