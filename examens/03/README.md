# Examen 3

> Remise : 28 mai\
> Pondération : 10 %

Votre tâche pour cet examen est de concevoir un programme qui affiche
dynamiquement la semaine en cours et le jour actuel dans un élément de
tableau. Un début de programme ainsi que des tests automatisés sont
inclus dans votre dépôt. Vous devez compléter le code et vous assurer
que les tests réussissent.

Il est interdit de modifier le code et le balisage fournis.

> [!NOTE]
> La méthode `getDay` d'un objet `Date` retourne le jour de la semaine
> de la date en question (0 correspondant à dimanche, 1 à lundi, et
> ainsi de suite).

## Instructions

1. Commencez par [créer un dépôt Classroom][Classroom] pour l'examen.
2. Ensuite, téléchargez le dépôt sur votre ordinateur (bouton _Code_
   vert → _Download ZIP_).
3. Pour soumettre votre travail, téléversez le fichier « index.js » (et
   seulement ce fichier) dans votre dépôt Classroom. Ne renommez pas le
   fichier, ne le placez pas dans un dossier, et ne le compressez pas.
