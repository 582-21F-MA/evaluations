# Examen 4

> Remise : 25 juin\
> Pondération : 10 %

Le [jeu de la taupe] (_Whack-A-Mole_ en anglais) est un jeu d'arcade où
il faut frapper le plus rapidement possible une taupe qui surgit
aléatoirement de trous. Votre tâche pour cet examen est de concevoir un
programme qui permet de jouer à une version simplifiée de ce jeu. Voici
les règles :

- Le jeu affiche neuf trous, chacun représenté par un bouton. Un trou
  aléatoire contient la taupe 🐹. Les autres trous sont vides.
- Lorsqu'on clique sur la taupe 🐹, elle se déplace vers un autre trou
  aléatoire.

Votre dépôt contient une fonction `createGame` que vous devez compléter.
Pour obtenir un nombre aléatoire, invoquez le paramètre `rand` de
`createGame`, qui correspond par défaut à [Math.random].

[jeu de la taupe]: https://en.wikipedia.org/wiki/Whac-A-Mole
[Math.random]: https://developer.mozilla.org/fr/docs/Web/JavaScript/Reference/Global_Objects/Math/random
